# @trippy/ui-kit

Touch-first UI components shared across the trippy web app and (later) Tauri
desktop / Capacitor mobile shells: knobs, faders, transport buttons, meters,
keyboard, slice grid.

**Status:** placeholder. Components currently live inline in
`apps/web/src/ui/` so they can iterate without a publish step. They'll graduate
into this package once their APIs settle and a second surface (Tauri) needs
them.

See [the plan](../../docs/) for the full roadmap.
